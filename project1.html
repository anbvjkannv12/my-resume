程式設計課

從大一到大二，我們修習了三門程式設計課程，循序漸進地打下了穩固的程式設計基礎。起初，我們從相對易上手的 Python 開始學習，待對語法與邏輯熟練後，進一步接觸結構更嚴謹的 C 語言，最後再進入功能更強大的 C++。這一連串的學習過程，不僅讓我逐步累積了對各類程式語言的理解，也強化了我解決問題與邏輯思維的能力。 當然，學習的過程中也少不了挑戰。例如，曾經看不懂一段程式碼、找不到錯誤的原因，甚至不知道該如何下手解決一題。面對這些困難，我一方面積極尋求老師的協助，另一方面也透過反覆思考與練習，最終逐一克服了這些問題。這些經驗不僅提升了我的技術能力，也讓我更加有信心面對未來更具挑戰性的程式開發任務。

3D設計與實作

這門課程主要透過專業的三維建模軟體 Maya，讓我們系統性地學習物件建模的原理與技術操作。課程中，我們不僅熟悉了基本介面與建模流程，還透過大量實作練習，逐步累積建模技巧與創作經驗。透過一系列實務作品的訓練，我更深刻掌握了從簡單幾何形狀到複雜結構的建模方法。在學期末，我也成功考取了 Maya 國際認證，這不僅驗證了我對該軟體的操作熟練度，也為我在數位內容設計與 3D 建模領域奠定了專業基礎。

Unity實作

在這次的程式設計課程中，我有機會首次接觸並實際操作 Unity 這套遊戲開發引擎。起初對 Unity 感到陌生，但在老師的教學與一連串實作練習的引導下，我逐漸掌握了它的基本功能與操作方式。 課程從最基礎的介面認識開始，接著我們實作了多個簡易遊戲專案。透過撰寫腳本，我學會如何將邏輯思維轉化為實際的程式碼，同時也理解了除錯與測試在開發流程中的重要性。 其中，我認為最有趣的部分，是將腳本應用在角色身上，讓角色能透過鍵盤輸入進行移動，甚至在碰觸特定物件時產生互動反應。這些互動效果不僅增添了遊戲的趣味性，也讓我真正體會到遊戲開發的成就感與樂趣。

Web程設作品

這門課程讓我初次接觸到用於網頁開發的程式語言——PHP，並學習如何將 PHP 與 HTML 結合，賦予網頁動態生成內容的能力。過去我對網站的認識，多停留在靜態的文字與排版設計，但透過這次的課程，我才真正了解，原來網頁可以具備互動性，能夠實作出像購物網站或使用者表單這樣的功能。
課堂上，老師先從 PHP 的基本語法教起，接著帶領我們進行多項實作練習，包含如何透過表單接收並處理使用者輸入。在這個過程中，我也遇到不少挑戰，例如因為少打分號或語法錯誤而導致程式無法執行。雖然這些錯誤讓我一度感到挫折，但透過不斷嘗試與修正，我逐漸掌握了 PHP 的語法邏輯，也提升了自己解決問題的能力。
