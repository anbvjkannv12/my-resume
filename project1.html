Unity實作

在這次的程式設計課中，我有機會接觸並實際操作 Unity 這套遊戲開發引擎。這是我第一次使用 Unity，起初感到有點陌生，不過透過老師的教學和實作練習，我逐漸理解了它的基本功能與操作方式。 我們從最基本的介面認識開始，例如 Scene、Game、Hierarchy、Inspector、Project 等面板的用途。接著，我學會了如何建立物件、加入元件（Components），例如剛體（Rigidbody）與碰撞器（Collider），並透過簡單的 C# 腳本來控制物件的移動與互動。 透過這些練習，我學到的不僅是 Unity 的操作，更加深了我對物件導向程式設計的理解。在撰寫腳本時，我體會到如何將邏輯轉化為程式碼，也學會了除錯與測試的重要性。 其中，我覺得最有趣的部分是將腳本套用在角色上，讓角色能夠透過鍵盤輸入移動，甚至碰到特定物件後產生反應。這讓我感受到開發遊戲的樂趣與成就感。 總結來說，這次的 Unity 軟體練習讓我收穫很多，也讓我對遊戲開發產生了濃厚的興趣。希望未來能夠學到更多進階的功能，例如動畫控制、場景切換與 UI 設計，持續提升自己的能力。


Web程設作品

這一堂課我們認識了 PHP 這個用於網頁開發的程式語言，並學習了如何將 PHP 與 HTML 結合，讓網頁具備動態生成內容的能力。過去我對於網頁的印象大多停留在靜態畫面，但透過這次的課程，我了解到後端程式語言如 PHP 能夠讓網站變得更有互動性，也能依照使用者的需求即時產生對應的內容。 課堂中，我們從基本的 PHP 語法開始，例如變數的宣告、條件判斷（if/else）、迴圈（for/while）等，逐步進入到如何在 HTML 中嵌入 PHP 程式碼。實作部分，我們練習了如何用表單（form）接收使用者輸入，並使用 PHP 處理輸入資料，例如計算數字、顯示歡迎詞等。這讓我更能理解前端和後端之間的互動流程。 在學習過程中，我也遇到一些挑戰，例如變數命名錯誤或是漏寫分號導致錯誤訊息出現，這些錯誤雖然讓我一開始有些困惑，但在反覆的嘗試與修正後，我對 PHP 的語法邏輯有了更清楚的認識。 整體而言，這堂課不僅讓我初步了解了 PHP 的基本應用，也讓我對網頁開發的流程有了更完整的認識。我期待未來能學習更多 PHP 進階功能，例如資料庫連接（MySQL）與會員系統的設計，進一步提升自己的網頁開發能力。
